# Автоматическое тестирование HamsterClicker

## Запуск авто-тестов

Откройте файл `test-runner.html` через Live Server.

## Что тестируется

### Storage модуль
- Сохранение данных в localStorage
- Загрузка данных из localStorage
- Игра запускается без сохранений

### Game логика
- Новая игра начинается с нуля
- Клики увеличивают монеты
- Загрузка сохраненного прогресса

### UI функции
- Форматирование отображения монет

## Структура тестов

Тесты написаны без внешних зависимостей, используя простую систему тестирования:
- `TestRunner.test(name, fn)` - регистрация теста
- `TestRunner.assert(condition, message)` - проверка условий
- `TestRunner.run()` - запуск всех тестов
