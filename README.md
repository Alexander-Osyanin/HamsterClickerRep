# Версия 3.0 - Пассивный Доход

## Задачи:
- Пассивные генераторы монет
- Автоматический заработок в секунду
- Несколько типов генераторов

---

## Инструкция по установке и запуску игры:

1. Заходите в репозиторий HamsterClicker

---

2. Нажимаете на зелёную кнопку "code" и там копируете ссылку на репозиторий

![guide1](img/guide/guide1.png)

---

3. В терминале вашего редактора кода пишете команду `git clone https://github.com/Alexander-Osyanin/HamsterClicker`

![guide2](img/guide/guide2.png)

---

4. В редакторе кода устанваливаете расширение по типу "Live Server" для открытия локального сервера

![guide3](img/guide/guide3.png)

---

5. Нажимаете правой кнопкой мыши на файл `index.html`, Затем нажимаете на "Open with Live Server" 

![guide4](img/guide/guide4.png)

---

Готово! Игра откроется в браузере установленном у вас по умолчанию.

---

# Автоматическое тестирование HamsterClicker v3.0

## Запуск авто-тестов

Откройте файл test-runner.html через Live Server.

### Список тестов:

1. **Игра начинается с корректных значений** - проверка начального состояния
2. **Клик увеличивает монеты** - базовая механика кликов
3. **Успешная покупка генератора** - покупка генератора при достаточных средствах
4. **Неудачная покупка генератора при недостатке монет** - защита от покупки генератора
5. **Суммарный доход от нескольких генераторов** - корректный подсчёт пассивного дохода
6. **Сохранение данных с генераторами** - сохранение прогресса генераторов
7. **Множитель увеличивает доход от клика** - работа множителя
