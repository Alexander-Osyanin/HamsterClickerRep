## Инструкция по установке и запуску игры:

1. Заходите в репозиторий HamsterClicker

---

2. Нажимаете на зелёную кнопку "code" и там копируете ссылку на репозиторий

![guide1](img/guide/guide1.png)

---

3. В терминале вашего редактора кода пишете команду `git clone -b (название нужной ветки) https://github.com/Alexander-Osyanin/HamsterClickerRep`

![guide2](img/guide/guide2.png)

---

4. В редакторе кода устанваливаете расширение по типу "Live Server" для открытия локального сервера

![guide3](img/guide/guide3.png)

---

5. Нажимаете правой кнопкой мыши на файл `index.html`, Затем нажимаете на "Open with Live Server" 

![guide4](img/guide/guide4.png)

---

Готово! Игра откроется в браузере установленном у вас по умолчанию.

# Автоматическое тестирование HamsterClicker v4.0

## Запуск авто-тестов

Откройте файл test-runner.html через Live Server.

### Список тестов:

1. **Игра начинается с корректных значений** - проверка начального состояния
2. **Клик увеличивает монеты и статистику** - базовая механика + счётчики
3. **Успешная покупка улучшения увеличивает счётчик покупок** - интеграция статистики
4. **Неудачная покупка при недостатке монет** - защита от покупки
5. **Начальный уровень — Новичок** - система уровней
6. **Уровень повышается при достижении порога** - переход на новый уровень
7. **Прогресс до следующего уровня считается корректно** - расчёт прогресс-бара
8. **Достижение разблокируется при выполнении условия** - система достижений
9. **Достижение не разблокируется повторно** - защита от повторной выдачи
10. **Сохранение данных v4.0** - сохранение всех новых полей
